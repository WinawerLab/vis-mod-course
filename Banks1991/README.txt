This is the readme file for the Banks1991 folder

This repository is Matlab based and uses the ISETBio toolbox (www.github.com/isetbio/isetbio).

There is a ToolboxToolbox registration file in the forked Winawerlab ToolboxRegistry called 'VisualModelingGradCourse'. If you use the ToolboxToolbox, you can type tbUse('VisualModelingGradCourse') to add this repository to your paths.
